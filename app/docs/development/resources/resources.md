# Developer Resources

Curated links to Bitcoin development tools, node software, hardware, explorers, and APIs. For libraries and code examples, see [Libraries & SDKs](/docs/development/libraries).

## Node Software

- **[Bitcoin Core](https://bitcoincore.org)** – Reference implementation; run a full node and use the RPC API.
- **[Knots](https://bitcoinknots.org)** – Bitcoin Core with additional patches and features.

## Node Hardware

- **[Start9](https://start9.com)** – Hardware and software for running nodes at home.
- **[Nodl](https://www.nodl.it)** – Plug-and-play Bitcoin (and Lightning) nodes.

## Explorers & APIs

- **[Mempool.space](https://mempool.space)** – Block explorer, API, and fee estimates.
- **[Blockstream Explorer](https://blockstream.info)** – Block explorer and API.
- **[Bitcoin Core RPC](https://developer.bitcoin.org/reference/rpc/)** – Built-in RPC when running a node.

## Utilities

- **[BitcoinDev Tools](/tools)** – CLI terminal, Stack Lab, Block Visualizer, Hash tool.
- **[Learn Me A Bitcoin – Tools](https://learnmeabitcoin.com/tools)** – Hashing, keys, transaction builder, and more.

## Further Curation

For a larger directory of Bitcoin and Lightning resources (by category and language), see [BitcoinDev.org](https://bitcoindev.org).
