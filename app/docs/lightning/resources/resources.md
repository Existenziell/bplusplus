# Lightning Resources

Curated links to Lightning Network tools, LSPs, dashboards, and routing/liquidity resources. For protocol docs, see [Lightning Network](/docs/lightning) and the [BOLT specs](https://github.com/lightning/bolts).

## LSPs & Services

- **[Lightning Network+ (LNG)](https://lightning.engineering)** – Enterprise LSP and infrastructure.
- **[Voltage](https://voltage.cloud)** – Cloud Lightning node hosting and LSP.
- **[LNBits](https://lnbits.com)** – Free, self-hostable Lightning wallet and extension server.

## Dashboards & Monitoring

- **[Amboss](https://amboss.space)** – Lightning network explorer and node analytics.
- **[1ML](https://1ml.com)** – Lightning network statistics and node search.
- **[Mempool.space – Lightning](https://mempool.space/lightning)** – Lightning graph and channel view.

## Routing & Liquidity

- **[Channel liquidity](https://github.com/lightning/bolts/blob/master/02-peer-protocol.md)** – BOLT 2 defines channel and liquidity semantics.
- **[Submarine Swaps](https://docs.lightning.engineering/lightning-network-tools/submarine-swaps)** – On-chain to Lightning and back.

## Learning & Development

- **[Lightning Dev Kit (LDK)](https://lightningdevkit.org)** – Library for embedding Lightning in apps.
- **[Core Lightning](https://github.com/ElementsProject/lightning)** – C implementation of the Lightning protocol.
- **[LND](https://github.com/lightningnetwork/lnd)** – Go implementation of the Lightning protocol.

For a larger directory, see [BitcoinDev.org – Lightning](https://bitcoindev.org/#lightning).
